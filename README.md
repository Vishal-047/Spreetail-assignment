# Shared Expense App

A robust shared expense application for managing shared costs, such as flatmate living expenses, trips, and group activities.

## Phase 1: Database Design & Documentation

This project handles complex expense sharing scenarios, including multiple payers, varying currencies, and changing group memberships.

## Setup Instructions

1. **Prerequisites:**
   - Node.js (v18+)
   - PostgreSQL (v14+)
2. **Database Setup:**
   - Create a new PostgreSQL database.
   - Run the initial schema migration using the provided `schema.sql` file.
3. **Installation:**
   - Clone the repository.
   - Run `npm install` to install dependencies.
4. **Environment Variables:**
   - Copy `.env.example` to `.env` and fill in your database credentials.
5. **Running the App:**
   - Run `npm run dev` to start the development server.

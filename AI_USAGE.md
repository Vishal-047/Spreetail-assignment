# AI Usage Documentation

This project utilized AI assistance during development.

## 1. Code Generation
* AI was used to scaffold the Next.js application, including the setup of TailwindCSS and shadcn/ui.
* The Prisma schema was iteratively designed with AI assistance to ensure robust relational integrity and support for complex business logic (e.g., event-sourced membership).

## 2. Business Logic Implementation
* The core `BalanceEngine` (debt simplification algorithm) and `AnomalyDetectionEngine` were drafted with AI assistance.
* AI was utilized to write comprehensive unit tests ensuring mathematical correctness and edge-case handling.

## 3. Review and Refactoring
* The AI acted as a simulated Code Reviewer to enforce maintainability and performance best practices.
* Generated code was reviewed to avoid "AI-style generic boilerplate" and ensure it remained understandable and maintainable.
